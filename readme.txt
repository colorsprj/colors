■ これは何？

　このアーカイブは、「伺か」「SSP」「CROW」「偽林檎」「ninix-aya」等の
デスクトップマスコットプレーヤー上で動作する、「ゴースト」と呼ばれる仮
想人格を組み込んだデスクトップマスコットです。上記のシステム上でお使い
下さい。
　このゴースト「COLORS」は、単独でゴースト用の外見ファイル「シェル」、
着せ替えパーツ詰め合わせ「パーツパック」、ゴースト自身を作成できます。
用意されたパーツを自由に組み合わせ、自分の好みのシェルを作成し、COLORS
や他のゴーストに組み込むことができます。さらに、パーツの追加も可能です。
