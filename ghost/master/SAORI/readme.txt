■ 機能

・new
	新しく画像を作成して画像 ID を返します

	引数 : 横幅, 高さ


・load
	PNG を読み込んで画像 ID を返します

	引数：ファイル名


・save
	PNG として保存します

	引数：画像ID, ファイル名


・dup
	画像を複製して画像 ID を返します

	引数：画像ID


・clear
	今までに読み込んだ画像をすべて破棄します
	unload 時には自動で解放されるので、終了時に呼び出す必要はありません

	引数：なし


・draw
	元画像の上に別画像を合成します
	各画像のアルファチャンネルを考慮して合成します

	引数：元画像の ID, 合成する画像の ID, X 座標, Y 座標, 透明度（百分率で指定。省略可能）


・fill
	画像を特定の色で塗りつぶします

	引数 : 画像 ID, 塗りつぶす画素値 (0xAARRGGBB)


・pixel
	指定された位置の画素値を設定・取得します
	画素値は 0xAARRGGBB 形式で表現された 10 進数で返ります

	引数：画像 ID, X 座標, Y 座標

	画素値を設定するときには引数の数が変わります

	引数：画像 ID, X 座標, Y 座標, 新しい画素値


・repaint
	ある色を別の色で置き換えます

	引数：画像ID, 元の色 (0xRRGGBB), 新しい色 (0xRRGGBB)


・tone
	全ピクセルの RGB 値に対して加算します
	オーバーフローは 255、アンダーフローは 0 に丸められます

	引数：画像 ID, R の値, G の値, B の値


・cut
	画像の一部分を切り出して、新しい画像IDを返します

	引数：画像 ID, X 座標, Y 座標, 横幅, 高さ


・resize
	画像を指定されたサイズに拡大縮小し、新しい画像IDを返します
	メソッドには
		"fast", "quality", "nearest_neighbor", "bilinear", "bicubic", "lanczos2", "lanczos3"
	のいずれかを指定してください

	引数：画像ID, メソッド, 横幅, 高さ

	※ 横幅、高さのどちらかを 0 にすると比率固定で縮小します

	拡大率を指定する場合は

	引数：画像ID, メソッド, 拡大率（百分率で指定）

	・メソッドの特徴
		nearest_neighbor : 最速だが平滑処理が行われないのでガタガタになる
		bilinear (=fast) : bicubic の 2 倍以上で処理が可能。エッジなどが僅かにぼやける
		bicubic (=quality) : bilinear より低速だが、鮮明な画像が得られる
		lanczos2 : bicubic より縮小においては高画質。処理は遅い
		lanczos3 : lanczos2 よりもさらに高画質。処理はさらに遅い
		lanczos4 : lanczos3 よりもさらに高画質。処理はさらにさらに遅い


・size
	指定された画像のサイズを取得します

	引数：画像ID


・opacity
	指定された画像の透明度を変更します

	引数：画像ID, 透明度（百分率で指定）
